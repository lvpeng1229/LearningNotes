### 01.题目要求
- 问题如下所示：
    - 计算从1+2+3+…+N的和
- 示例 :
    ```
    计算从1+2+3+…+100的和是5050
    ```

### 03.实例代码
- 如下所示
    ```
    /**
     * 获取从1+到N的和
     *
     * @param num
     * @return
     */
    public static int getSum(int num) {
        if (num == 1) {
            return 1;
        }
        return num + getSum(num - 1);
    }
    ```
- 测试代码
    ```
    public static void main(String args[]) {
         System.out.println(getSum(100));
    }
    
    //执行结果
    5050
    ```
